## SocialMediaWebsite 📢

# A full-stack social media web application developed as a coursework project for a Software Engineering course. The platform allows users to authenticate via Google and interact with a shared message board — viewing, creating, and deleting posts.

## 🎯 Objective
# The objective of this project was to design and build a functional, full-stack web application that demonstrates core software engineering principles, including RESTful API design, database integration, user authentication, and cloud deployment. The application serves as a lightweight social media platform where authenticated users can participate in a shared message feed, fostering an understanding of modern web development workflows from front-end to back-end to deployment.

## ✨ Features

# Google OAuth Authentication — Sign in securely using your Gmail account via Google OAuth 2.0. No separate registration required.
# View Messages — Browse all messages posted by users on the platform in a clean, accessible feed.
# Create Messages — Authenticated users can compose and publish new messages to the shared board.
# Delete Messages — Users can remove their own messages from the platform.
# Persistent Storage — All messages are stored in a PostgreSQL database, ensuring data persists across sessions.
# Cloud Deployment — The application is deployed via Dokku, making it accessible from anywhere.
